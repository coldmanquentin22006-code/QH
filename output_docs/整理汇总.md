# 7所幼儿园联系单/确认单整理汇总

（东平幼儿园本轮不处理；石壁镇中心幼儿园＝南俸幼儿园；塔洋镇中心幼儿园＝千秋幼儿园；万泉镇中心幼儿园＝东升幼儿园；大路镇中心幼儿园东红分园＝东红幼儿园，均为使用单位名称与确认单校名不一致的情况，已在下方各校说明中注明）

## 一、总体处理规则

1. 联系单里有、但确认单里没有的内容 → 视为该内容已取消，在确认单对应位置备注"已取消，见联系单一第X条"。
2. 确认单里有、但联系单里没有的内容 → 补充进新做的《工作联系单（二）》，并在确认单该条备注"对应工作联系单二第X条"。
3. 材质/规格类差异（比如围墙材质、护栏类型、立柱高度这种"做法优化"性质的不同）→ 备注"经设计单位优化调整"，不追究具体原因。
4. 变压器容量（KVA）、高压计量设备这类"设备参数" → 备注为"设备参数，不纳入本次工程量确认范围"。
5. 结果性冲突（两份文件说的是完全不同的东西，不是规格优化）→ 逐项跟需求方确认后采纳一方，或标注"需现场核实"。

新确认单的条目顺序统一按《联系单一》的顺序排列，联系单一签字盖章内容不改，有差异都在确认单一侧加注说明。
